Open Source

Dans notre logique de communs numériques, Nudger est opensource et opendata. Rendre notre code disponible pour tous permet de garantir la transparence de l'écoscore et de faire de ce comparateur écologique une aventure collective.

Nous aider !

Qu'il soit plutôt sur le back ou sur le front, nous sommes preneurs de toute contribution ! La première étape est sans doute alors de faire tourner le projet nudger (open4goods) directement sur votre machine, la procédure est décrite sur notre dépôt Github. Certaines tâches (issues) sont spécialement adaptées pour une première intervention sur le projet.

Comment contribuer ?

Si vous souhaitez simplement comprendre ou auditer le fonctionnement du site vous pouvez vous laisser guider simplement dans les documentations ci-dessus. Si vous souhaitez contribuer, tout fonctionne par Pull Request. C'est plutôt réservé aux Geeks mais cela reste abordable à tous !

Enfin, une belle façon de nous aider est aussi de d'utiliser Nudger, d'en parler autour de vous et de nous faire retour de vos idées directement sur le dépôt github ou via notre outil de feedback ! Enfin, nous serons heureux d'échanger avec vous de façon plus traditionnelle via notre formulaire de contact si vous le souhaitez !
