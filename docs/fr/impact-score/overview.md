# Vue d’ensemble de l’Impact Score

L’**Impact Score** est notre note environnementale pour les produits
électriques et électroniques. Il agrège plusieurs critères en un score unique
**clair, comparable et transparent**.

## Principes clés

- **Échelle normalisée** de 0 à 5.
- **Pondération transparente** spécifique à chaque catégorie.
- **Pénalité de qualité de donnée** lorsqu’une information manque.

## Où le score apparaît

Vous verrez l’Impact Score sur les pages produit, les pages catégorie et dans
la section méthodologie. Pour aller plus loin, explorez les documents associés.
